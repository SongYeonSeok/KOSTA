- 서버 통신 방법 : server <-> client
- client -> server , server -> client : 이를 핸드쉐이크라고 한다.
- client -> server 과정에서, client에서 보내주는 정보를 검증한 후에 server에서 반응해서 결과를 보내준다.
- 그리고 통신하는 데 있어서 정해진 약속인 프로토콜을 준수한다. 이러한 규정에 의해 검증하고 확인하는 과정을 거친다.
