- Read & Write (명령어)
  - 중요한 키 : digital (0(off)과 1(on))
    - 디지털에서의 1과 0은 일반적인 값이 아닌 상태값이다. 1 : On, 0 : Off
    - data => 1bit / 4bit ... (8bit가 일반적으로 사용된다.)
  - 직렬 vs 병렬
    - 직렬 : 하나의 선(bit)으로 0과 1로 파형이 구성되어 있는 것
    - 병렬 : 여러개의 선(bit)로 0과 1로 파형이 동시에 표현되어 있는 것. (보통 1byte = 8bit => 8개의 선을 사용)
